Downline - down to earth markup


Headers

Headers are denoted using a # symbol, the amount of #'s before a line of text represents the size of header it will be.

Lists

Lists items are denoted with a - symbol, sequential list items will be grouped into a list.

Links

Links are denoted with a [Link text](URL) format, where "Link Text" is the text which appears and URL is the link the text should point to.

Bold/Italics

Bold is denoted with a double asterisc (**) at the beginning of a string and another set of asteriscs at the end of the string. Italics is the same but with only a single asterisc (*)


Why Downline?

Downline implements a strict subset of the Markdown syntax designed to reduce ambiguity, improve readability and speed up parse time.
