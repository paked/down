Down - down to earth markup


Headers

In Down headers come in 3 sizes. Big, middle and small. Headers are denoted by the # symbol, the more #'s you add the smaller a header you get.

Lists

Lists are shown by the * characters prefixing a line of text.

Bold

Bold text is created by wrapping the targetted text in ** symbols

Italics

Italicised text is created by wrapping targetted text in * symbols.


Why Downline?

Downline implements a strict subset of the Markdown syntax designed to reduce ambiguity, improve readability and speed up parse time.
